The design and the application of the solar-thermal control system

Overview

This is a project aiming in control a solar-thermal disk to track the sun so that it could generate power at its max efficiency. In short, our goal is to let the figure \ref{mylabel} change to image 2.

Experiment

Open-loop Control Algorithm

Here we use a open-loop algorithm and a close-loop one to control the disk.
The former took the altitude, longitude and the time to calculate the real-time position of the sun. The max error of this algorithm is up to 3°. Here we only wants it to track the sun roughly and automatically.

Close-loop Control Algorithm

This algorithm is to lower down the error angle between the sun light and the normal of the disk further. We used a four-quadrant sensor which could send out the diff-angle between the sun light and normal of itself. So the key point here is to make sure the sensor is mounted exactly with its normal parallel to disk’s. We use a tricky method:

  • Mount the sensor onto the disk roughly.
  • Run a control algorithm like bang-bang controller.
  • Adjust the mounted angle continuously until the disk could almost focus all of the sun light.

In this way, only when the sensor’s normal is parallel to the disk’s then the disk would have a smallest and brightest focus on the absorber.
