Self-Balance Bike with Inverted Pendulum

This is a project using the inverted pendulum with the inertia wheel to keep the bike balance.

Step 1

At first, I was going to build up an inverted pendulum, but with some special feature, like using the inertia wheel. It has three parts: pendulum, motor with driver and controller.

Step 2

For pendulum, it was built by the carbon tube which is often used in the drones. A bearing from a motor was mounted in the bottom of the pendulum, which made sure the pendulum can swing itself up without too much friction.

For the motor part, it was the second difficult part in this project. To control this pendulum, I needed to control the torque that the motor generate in order to provide force to control the pendulum. As a result, I used the FOC motor driving algorithm. The specific solution is using the motor control solution(F28027) from TI.

Step 3

Last but not least, I developed a multi-loop control algorithm to make sure the inverted pendulum can not only keep balance but also swing by itself.
